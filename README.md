# Ghép Trang Đôi Manga

URL: https://wbcutter.github.io/merge/

## Mô tả
Công cụ web để ghép các trang đơn thành trang đôi (double spread) phục vụ biên tập và đọc manga/truyện tranh. Hoạt động hoàn toàn trên trình duyệt, không cần cài đặt.

## Tính năng chính
- Kéo & thả ảnh (hoặc click để chọn) để thêm ảnh vào danh sách xử lý.
- Hỗ trợ định dạng: JPG, PNG, WebP.
- Ghép hai ảnh liền kề thành một trang đôi.
- Chỉnh chất lượng ảnh đầu ra (ví dụ: 100%, 99%, 98%).
- Nút chức năng: Ghép tất cả, Tải tất cả, Chọn tất cả / Bỏ chọn, Xóa tất cả.
- Giao diện đơn giản, trực quan, dễ dùng cho người mới.

## Hướng dẫn sử dụng
1. Truy cập: `https://wbcutter.github.io/merge/`
2. Kéo thả hoặc chọn file ảnh (JPG / PNG / WebP).
3. Kiểm tra thứ tự ảnh (thứ tự ảnh quyết định cặp ghép).
4. Chọn chất lượng xuất (nếu có).
5. Nhấn **Ghép tất cả** để tiến hành ghép.
6. Nhấn **Tải tất cả** để download ảnh đã ghép hoặc tải từng ảnh theo nhu cầu.
7. Dùng **Xóa tất cả** nếu muốn bắt đầu lại.

## Lưu ý
- Thứ tự ảnh quan trọng: ảnh được ghép theo thứ tự hiển thị trong danh sách.
- Nếu muốn ghép trang cụ thể theo cách khác, sắp xếp lại thứ tự ảnh trước khi ghép.
- Xử lý nhiều ảnh hoặc ảnh kích thước lớn có thể tiêu tốn bộ nhớ và thời gian trên trình duyệt.
- Chỉ hỗ trợ các định dạng nêu trên; các định dạng khác cần chuyển đổi trước khi dùng.

## Ứng dụng thực tế
- Chuẩn hóa trang để đọc theo dạng spread.
- Chuẩn bị ảnh cho EPUB / CBZ / trình đọc truyện.
- Tạo preview trang đôi cho web hoặc tài liệu in.

## Gợi ý cải tiến (không bắt buộc)
- Thêm tuỳ chọn lề/khung trắng giữa hai trang khi ghép.
- Hỗ trợ kéo-thả theo thư mục, tự nhận thứ tự file theo tên hoặc metadata.
- Tùy chọn ghép theo hướng trái→phải hoặc phải→trái (hữu ích cho manga đọc phải→trái).
